# Projects

This folder is the team's shared project workspace. Each project has its own folder.

---

## How to use

**Starting a new project:**
1. Copy `_Project Template/` → rename to your project name
2. Fill in `Overview.md` — owner, status, definition of done
3. Rename `Contributor.md` → `[YourName].md`
4. Share the folder path with anyone else on the project

**Working on a project:**
- Edit **only your own `[Name].md` file** for tasks and notes (avoids sync conflicts)
- Update `Overview.md` status when something material changes
- Log decisions and updates in `Log.md` (add to top, newest first)

**Completing a project:**
- Check off all Definition of Done items in Overview.md
- Set Status → Complete
- Move folder to `Projects/_Archive/` (create if needed)

## Rules

- One folder per project
- Per-person files for tasks/notes — do not all edit the same file
- Log.md for decisions — if it was discussed and decided, it goes here
- Overview.md is the source of truth — keep it current
