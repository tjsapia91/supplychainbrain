## How to take a clean screenshot on Windows

1. Press **Windows Key + Shift + S** → opens Snipping Tool
2. Drag a rectangle around the area you want
3. Click the notification → click Save → save as PNG to this folder
4. Use the exact filename from the table above

Or: **PrtScn** key → paste into Snipping Tool / Paint → save as PNG.
